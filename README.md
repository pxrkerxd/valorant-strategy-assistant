# 🎯 Valorant Strategy Assistant

A sleek, glassmorphic web application designed for Valorant players to optimize their tactical gameplay. This tool provides map-specific agent recommendations, detailed agent counters, and an interactive **Team Comp Rater**.

## ✨ Features

* **Map Intel**: Strategic overviews and agent tier lists for competitive maps.
* **Agent Data**: Detailed breakdown of every agent's strengths and weaknesses.
* **Role Filtering**: Easily sort agents by their role (Duelist, Controller, etc.) to find the right fit for your playstyle.
* **Interactive Team Builder**: Select a 5-man roster and receive real-time tactical feedback on your team's composition.
* **Glassmorphic UI**: A modern, frosted-glass interface inspired by the official Valorant aesthetic.

## 🛠️ Tech Stack

* **Frontend**: HTML5, CSS3 (Flexbox, Grid, Animations)
* **Logic**: Vanilla JavaScript (ES6+)
* **Data**: JSON (External data management)
* **Design**: Glassmorphism, CSS Keyframe Animations

## 📖 How to Use

1.  **Map Intel**: Click the "Map Intel" button to see which agents dominate specific maps.
2.  **Agent Data**: Use the "Agent Data" section to learn how to play as or against specific agents. Use the dropdown to filter by role.
3.  **Team Builder**: Navigate to "Team Builder," click on 5 agents to fill your slots, and hit **Analyze Composition** to see your team's tactical rating.
